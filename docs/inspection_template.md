# Flow Radar 人工抽检标注模板

> 用于 72h 验证期间的信号质量评估

---

## 抽检规则

- 抽样数量: N=30 (或总信号的 10%，取较大值)
- 抽样方法: 分层随机 (ACTIVITY 和 CONFIRMED 各占一半)
- 评判时限: 信号后 15 分钟内的价格走势

---

## 评判标准

| 等级 | 判定 | 标准 |
|------|------|------|
| **命中** | HIT | 价格向预期方向移动 >= 0.3%，或明显持续吃单 |
| **未命中** | MISS | 价格反向移动 >= 0.5%，或信号后无明显成交 |
| **不确定** | UNCERTAIN | 价格波动 < 0.3%，无法明确判断 |

---

## 汇总统计

| 指标 | 保守计算 | 中性计算 |
|------|----------|----------|
| 样本数 N | | |
| HIT 数量 | | |
| MISS 数量 | | |
| UNCERTAIN 数量 | | |
| **Precision** | HIT / (HIT + MISS) | HIT / N |
| CONFIRMED 命中率 | | |
| ACTIVITY 命中率 | | |

> - 保守计算: 排除 UNCERTAIN，只算 HIT / (HIT + MISS)
> - 中性计算: 包含 UNCERTAIN，算 HIT / 总数
